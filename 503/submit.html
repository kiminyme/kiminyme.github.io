Submitting your work

Once you have completed the work, you will submit the entire YourNameHTML folder to your instructor following the instructions below:

  • Make sure that all changes to the HTML and CSS files have been saved, and close all open files.
  • Find the YourNameHTML folder in the Computer window. Open the file to verify that it contains the following files:
    • introduction.html
    • picture.jpg
    • style.css
  • If your folder includes any other files, you can delete them at this point. You may also want to open the HTML file in your browser to verify that it looks right one last time before submitting it.
  • Use the Computer Back arrow so that you can see the YourNameHTML folder itself, instead of the contents of the folder.
  • Right-click the folder. Point to Send To in the Shortcut menu, then click on Compressed (Zipped) folder in the submenu.
  • This will create a new file named YourNameHTML, but with the icon for a zipped folder.
  • Submit the zipped folder to the dropbox for this assignment in the online course, following the instructions given by your instructor.

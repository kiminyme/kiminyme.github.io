Exercise Overview

Purchasing a computer system typically costs a significant amount of money, so in many ways, it should be considered an investment that will be used for several years, rather than done as a spur-of-the-moment decision. This WebQuest is set up to help students understand the steps that should be taken before purchasing any kind of computer system or similar technology.

Student Instructions

Work through the following sections to complete the WebQuest.

  • Introduction: An overview of the WebQuest
  • Tasks: A list of tasks that will be completed
  • Process: Details about how to complete the tasks
  • Evaluation: An opportunity to see how the work will be assessed, with the option of making revisions based on that information
  • Conclusion: A discussion of what should have been learned as a result of completing the Web Quest
